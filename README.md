# 🛍 Sistema de Gestión de Productos

Aplicación web desarrollada con Flask que permite gestionar productos mediante operaciones CRUD (Crear, Leer, Actualizar y Eliminar).  
Cuenta con interfaz moderna estilo glassmorphism y manejo dinámico de stock.

---

## 🚀 Características

- ✅ Crear productos
- 📋 Listar productos
- ✏ Editar productos
- 🗑 Eliminar productos
- 🖼 Subida de imágenes
- 📦 Control de stock dinámico
- 🎨 Diseño moderno con efectos visuales
- 💻 Interfaz responsive con Bootstrap

---

## 🛠 Tecnologías utilizadas

- Python
- Flask
- HTML5
- CSS3
- Bootstrap 5
- Jinja2
- SQLite (o la base de datos que estés usando)
